# 🔊 AI Gesture Volume Controller

A computer vision-based application that allows users to control the system volume using simple hand gestures. This project also integrates virtual mouse control and screen brightness adjustment into a single AI Gesture Control Suite.

---

## 📌 Features

### 🔊 Volume Controller
- Increase Volume
- Decrease Volume
- Live Volume Bar
- Real-time Gesture Detection

### 🖱️ Virtual Mouse
- Cursor Movement
- Left Click
- Right Click
- Double Click
- Drag & Drop

### 💡 Brightness Controller
- Increase Brightness
- Decrease Brightness
- Real-time Brightness Control

---

## 🛠️ Technologies Used

- Python
- OpenCV
- MediaPipe
- PyAutoGUI
- Pycaw
- Screen Brightness Control
- NumPy
